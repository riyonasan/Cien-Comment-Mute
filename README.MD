# Ci-en Comment Mute Userscript

Ci-en のコメント欄で特定ユーザーのコメントを非表示にする Tampermonkey/Greasemonkey 用スクリプトです。  
ミュート管理パネルと再表示機能を備えています。

## インストール

Tampermonkey をブラウザに導入した上で、以下のリンクをクリックしてください。  

[インストールする](https://github.com/yourname/ci-en-mute/raw/main/cien-comment-mute.user.js)

## 機能

- コメント横に「ミュート」ボタンを追加  
- ミュートしたユーザーのコメントを自動で非表示  
- ページ右下の「ミュート管理」パネルでミュート済みユーザーの確認・解除  
- 「ミュート再表示」ボタンで一時的に非表示コメントを再表示  

## 更新方法

Tampermonkey が自動で更新チェックを行います。  
手動で更新したい場合は Tampermonkey のダッシュボードから対象スクリプトを選び「更新」をクリックしてください。
